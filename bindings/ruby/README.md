Ruby bindings
====

## Descripton

Build library of Toolbit Interface for Ruby.


## Require

ruby  
  32bit version is recommended  

swig  
  http://www.swig.org/  


## Build

    cd swig
    ./build.sh
tbi.so will be created and copied to ../samples folder


## Test

    cd ../samples
    ruby chopper_test.rb
    ruby luke_test.rb
